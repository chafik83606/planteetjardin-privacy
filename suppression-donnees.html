Suppression des données

Plante & Jardin

Dernière mise à jour : 24 juin 2026

L'application Plante & Jardin stocke vos plantes, journal et photos uniquement sur votre appareil. Nous n'hébergeons pas de compte utilisateur ni de serveur centralisé pour ces données.

Comment supprimer vos données

Option 1 — Désinstaller l'application (recommandé)

La désinstallation de Plante & Jardin sur Android supprime les données locales de l'application (plantes, journal, photos enregistrées dans l'app).

  1. Ouvrez Paramètres → Applications
  2. Sélectionnez Plante & Jardin
  3. Appuyez sur Désinstaller

Option 2 — Effacer les données sans désinstaller

  1. Paramètres → Applications → Plante & Jardin
  2. Stockage → Effacer les données (ou Effacer le cache et les données)

Données envoyées à des tiers

Si vous avez utilisé l'identification ou le diagnostic IA avec une clé API OpenAI, les photos transmises sont traitées par OpenAI selon leur propre politique. Pour toute demande liée à ces données, consultez la politique OpenAI.

Délai de traitement

La suppression locale est immédiate dès que vous effacez les données ou désinstallez l'application.
